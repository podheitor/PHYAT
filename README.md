# PHYBRV - PodHeitor YouTube Bulk Related Video

A Chrome/Brave browser extension that allows YouTube creators to bulk-set the "Related Video" field on multiple videos simultaneously from the YouTube Studio interface.

**No API keys or Google Cloud setup needed!** The extension uses your existing YouTube Studio login session.

## Features

- 🎯 **Bulk Selection** — Select multiple videos using YouTube Studio's built-in checkboxes
- 🔗 **Set Related Video** — Apply a single related video to all selected videos at once
- 🔍 **Video Browser** — Search and browse your channel's videos in a beautiful modal UI
- 📊 **Progress Tracking** — Real-time progress indicator during bulk operations
- 🎨 **Native Dark Theme** — Seamlessly matches YouTube Studio's design language
- 🔒 **Zero Config Auth** — Uses your existing YouTube session, no API keys needed

## Installation

1. Clone or download this repository
2. Open `chrome://extensions` in Chrome or Brave
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked** and select this extension's directory
5. Navigate to **YouTube Studio** → **Content** → **Videos**
6. That's it! The extension is ready to use.

## Usage

1. Open **YouTube Studio** → **Content** → **Videos**
2. Select one or more videos using the **checkboxes**
3. Click the red **"Set Related Video"** button (bottom-right corner)
4. Browse or search your channel's videos in the modal
5. Click a video to select it as the related video
6. Click **"Apply Related Video"** to confirm
7. Wait for the extension to process each video
