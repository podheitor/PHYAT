How to use:

  1. Go to YouTube Studio → Content → Videos
  2. Select videos using the checkboxes
  3. Click the red "Set Related Video" button
  4. Browse and select the related video
  5. Confirm to apply to all selected videos
💡 Tip: No API keys or Google Cloud setup needed! The extension uses your existing YouTube Studio login session.
